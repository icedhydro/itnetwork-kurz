
   O aplikaci  		

***********************************************************************************************

   Aplikace byla vytvořena jako projekt k závěrečné zkoušce v rekvalifikačním kurzu Programátor WWW v C# .NET itnetwork.cz .

   Je postavena na jazyce C# a využívá technologii ASP.NET 7 Core MVC. Cílem projektu bylo implementovat kompletní správu (CRUD) pojištěnců, jejich pojištění a pojistných 	   událostí, s funkcionalitou uživatelských rolí (pojištěný, administrátor) pro definování přístupu k různým datům a podporou SQL databáze.

   Admin má oprávnění pro vytváření, editaci a mazání pojištěnců, pojištění a pojistných událostí. Uživatelé mohou měnit své nastavení a prohlížet (číst) pojištění a 	         pojistné události. Aplikace je naprogramována podle dobrých praktik, s plnou responzivitou.

   Jméno firmy, adresa, telefonní číslo a její pojištěnci uvedené v aplikaci jsou smyšlené a slouží pouze pro demonstrační účely.
